# CSS Mini Project: Balance Sheet

## Description

A simple project to practice using pseudo-selectors in CSS. This project is a simple webpage with a balance sheet.

## Features

**CSS**

-   Psuedo-selectors
-   Attribute selectors
-   Attribute value selectors
-   Child selectors
-   sr-only class
-   id selectors
-   class selectors
